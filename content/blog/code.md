# Code Blocks

Code blocks..with automatic syntax highlighting

## Code block with backticks

```html
<!doctype html>
<html lang="en">
<head>
  <meta charset="utf-8">
  <title>Example HTML5 Document</title>
</head>
<body>
  <p>Test</p>
</body>
</html>
```

## Line Numbers, Highlights
